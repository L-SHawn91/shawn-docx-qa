# Roadmap

shawn-docx-qa is an early-stage public-safe scaffold. APIs are unstable and examples intentionally use toy data or public metadata only.

## Near-term

- Keep the public boundary scanner passing.
- Add small tests and toy examples for the layer role.
- Improve documentation for contributors.
- Link changes back to the SHawn EvidenceMap flagship when they affect the public evidence workflow.

## Boundary

Do not add credentials, private corpora, PDFs, unpublished manuscripts, local paths, workflow logs, memory exports, or private project state.
